## Sample Run

| Language        | Steps                                                          |
| --------------- | -------------------------------------------------------------- |
| JavaScript      | `node [filename.js]`                                           |
| Python          | `python [filename.py]`                                         |
| C               | `gcc [filename.c]`<br>`./a.out  # unix`<br>`a.exe  # windows`  |
| CPP             | `g++ [filename.cpp]`<br>`./a.out # unix`<br>`a.exe # windows`  |
| Java            | `javac [filename.java]`<br>`java [filename]`                   |
